# Flikky — умные карточки для запоминания

**Flikky** — это приложение для изучения иностранных слов и других материалов с помощью карточек и интервальных повторений. Алгоритм напоминает FSRS, а интерфейс выполнен в тёмной теме с возможностью кастомизации (стеклянная нижняя панель, акцентные цвета). Приложение работает на Android и iOS.

## 🚀 Основные возможности

- **Создание колод и карточек**  
  Каждая карточка содержит слово/термин, перевод, транскрипцию, пример использования и перевод примера, а также тег для группировки.

- **Интервальные повторения**  
  Карточки распределяются по алгоритму, близкому к FSRS. Пользователь выбирает оценку (Забыл / Трудно / Хорошо / Легко), и система планирует следующий показ.

- **Режимы тренировки**  
  - **FSRS** — карточки, подлежащие повторению по алгоритму.  
  - **Новые** — только не изученные карточки.  
  - **Обратный режим** — показывает перевод и спрашивает слово.

- **Теги и фильтрация**  
  Каждой карточке можно назначить тег. При настройке тренировки можно выбрать один или несколько тегов.

- **Статистика**  
  Отслеживание количества новых и повторённых карточек за день, а также точность за последние 7 дней.

- **Импорт/экспорт**  
  - Импорт колод из CSV (формат: слово;пример;перевод_слова;перевод_примера;транскрипция;тег).  
  - Создание резервных копий всех данных (колоды, карточки, логи повторений, статистика) в JSON‑файлы.  
  - Восстановление из бэкапов, удаление и переименование бэкапов.

- **Настройки внешнего вида**  
  - Выбор языка (русский, английский).  
  - Включение/выключение «стеклянного» эффекта для нижней панели (размытие фона).  
  - Регулировка силы размытия.  
  *Цветовая кастомизация в разработке.*

- **Озвучка**  
  Для слов на китайском (и других языках) можно прослушать произношение (используется TTS).

- **Отмена последнего действия (Undo)**  
  Во время тренировки можно отменить последний ответ, чтобы исправить ошибку.

## 🛠 Стек технологий

- **Flutter** — кроссплатформенная разработка.  
- **Riverpod** — управление состоянием.  
- **Isar** — быстрая локальная NoSQL база данных.  
- **SharedPreferences** — хранение настроек.  
- **FilePicker + path_provider** — работа с файловой системой.  
- **intl** — форматирование дат и чисел.  
- **url_launcher** — открытие ссылок.  
- **share_plus** — отправка файлов.  
- **csv** — парсинг CSV.
